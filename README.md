# Backend II: Entrega Final - 

Este proyecto es un servidor desarrollado con **Node.js** y **Express**, utilizando una arquitectura profesional con separación de capas (Routes, Controllers, DAO, DTO).

## Tecnologías utilizadas
* **Node.js** con ES Modules (`type: module`).
* **Express** para el servidor web.
* **MongoDB & Mongoose** para la persistencia de datos.
* **Passport & JWT** para la autenticación basada en cookies.
* **Bcrypt** para el hasheo de contraseñas.

## Requisitos previos
Antes de correr el proyecto, asegúrate de tener instalado:
* [Node.js](https://nodejs.org/)
* [MongoDB](https://www.mongodb.com/try/download/community) (Local o Atlas)
